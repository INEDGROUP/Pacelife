# PaceLife

An iOS app for tracking your pace and lifestyle.

## Project Structure

```
PaceLife/
├── Core/
│   ├── Theme/          # Colors, typography, design tokens
│   ├── Extensions/     # Swift extensions
│   └── Utilities/      # Helpers and shared utilities
├── Features/
│   ├── Home/           # Home screen
│   ├── Map/            # Map and location features
│   ├── Checkin/        # Check-in functionality
│   ├── Insights/       # Analytics and insights
│   ├── Profile/        # User profile
│   ├── Onboarding/     # Onboarding flow
│   └── Subscription/   # Subscription management
├── Services/
│   ├── Supabase/       # Supabase backend integration
│   ├── AI/             # AI service integration
│   ├── Health/         # HealthKit integration
│   └── Weather/        # Weather service integration
├── Models/             # Data models
├── Components/         # Reusable UI components
└── Resources/
    └── Fonts/          # Custom fonts
```

## Requirements

- iOS 17+
- Xcode 16+
- Swift 5.10+
